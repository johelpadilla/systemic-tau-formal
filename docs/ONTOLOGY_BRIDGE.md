# Ontology bridge (semi-technical)

**Audience:** philosophers of science, ontologists, philosophical anthropologists.  
**Labels:** mostly `[AFIRMACIÓN ONTOLÓGICA]` and `[INTERPRETACIÓN FÍSICA]`.

## RECD as emergent discrete time

The RECD does not replace chronological clock time for logistics; it constructs a **second clock** that advances when relational ordinal structure reorganizes. Chronos remains the sampling index; Kairos is event-driven accumulation \(T = \sum g(\tau_s)\,\Delta t_k\).

## Four-level architecture (sketch)

| Level | Role | Avoids |
|-------|------|--------|
| L0 Observational | Rank series, windows | Treating numbers as being |
| L1 Metric | τₛ, g, Δtₖ | Pure narrative without measure |
| L2 Dynamical | Return maps, Feigenbaum universality **where theorems apply** | Inflating empirics into necessity |
| L3 Ontological | Claims about discrete extramental time / ascent | Collapsing into L1 equations |

## Baron’s Trilemma (infinite regress / circularity / dogmatism)

**`[AFIRMACIÓN ONTOLÓGICA]`** Claim of the programme: stratified levels prevent (i) justifying time only by deeper time (regress), (ii) defining the clock only by the clock (circularity), (iii) free stipulation without contact with L0–L2 (dogmatism).

This is **not** a Lean arithmetic theorem; `Ontology.lean` encodes the architecture as a formal **specification** open to critique (`philosophy-challenge` issues).

## Homology with Polo’s efficient causality

**`[AFIRMACIÓN ONTOLÓGICA]`** The programme proposes a *homology* (not identity) between:

- efficient causality as act that actualizes without being a further object in the series, and  
- RECD ticks as reorganizations that advance discrete time without being mere calendar increments.

**Explicit non-claims:** Feigenbaum’s \(\delta\) is not Polo’s cause; dengue data do not prove metaphysics.
